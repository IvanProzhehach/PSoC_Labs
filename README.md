

## **Lab 1: GPIO & Button Control**

### **Overview**

This project demonstrates basic work with **PSoC Creator™**, including project setup, pin configuration, and simple GPIO interaction.

### **Functionality**

* Configuration of digital input (button) and output (LED)
* Proper pin initialization and hardware connection
* Implementation of button-controlled LED logic

### **Result**

A simple embedded system where the LED is activated when the button is pressed.

## **Лаб. 1: GPIO та кнопка**

### **Опис**

Цей проєкт демонструє базову роботу з **PSoC Creator™**, включаючи створення проєкту, налаштування пінів та просту взаємодію через GPIO.

### **Функціональність**

* Налаштування цифрового входу (кнопка) та виходу (світлодіод)
* Коректна ініціалізація пінів та їх апаратне підключення
* Реалізація логіки керування світлодіодом через кнопку

### **Результат**

Проста вбудована система, у якій світлодіод вмикається при натисканні кнопки.
